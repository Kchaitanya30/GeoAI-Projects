# GIS-Based Prediction of Auto Theft & Carjacking Hotspots in Baltimore City

## Overview
This project uses GIS (Geographic Information Systems) and geospatial analysis techniques to predict and identify hotspots of auto theft and carjacking incidents in Baltimore City.

## Project Goals
- Analyze historical crime data to identify patterns
- Predict high-risk areas for auto theft and carjacking
- Provide actionable insights for law enforcement and community safety

## Contents
- Data analysis and visualization
- Geospatial modeling
- Predictive analytics
- Maps and heatmaps of hotspots

## Technologies
- GIS tools and geospatial libraries
- Python (GeoPandas, Folium, etc.)
- Data analysis and machine learning
